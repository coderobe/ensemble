# Ensemble

A build environment for [conductor](https://github.com/coderobe/conductor) ([libchoir](https://github.com/coderobe/libchoir))

## Usage 
- `make update` -> update build files 
- `make build` -> compile everything 
- `make test` -> run the testsuite 
- `make clean` -> clean up
